# Session Decision Summary: add golang.org/x/term dependency

Date: 2026-06-24 18:00
Base commit: 12e381b

## Context
Adding golang.org/x/term as a dependency for future raw terminal input support (keyboard accelerators like Ctrl+\ for model cycling).

## Changes Made
- go.mod: added golang.org/x/term v0.44.0 and golang.org/x/sys v0.46.0
- go.sum: new file with dependency hashes
- go directive bumped from 1.21 to 1.25.0 (required by x/term)

## Decisions And Rationale
- golang.org/x/term chosen over liner/readline for raw input: single lightweight dependency, cross-platform (Linux/macOS/Windows), official Go sub-repository, BSD-3-Clause, zero cgo.
- Go version bump to 1.25.0 accepted as acceptable trade-off for the dependency.
